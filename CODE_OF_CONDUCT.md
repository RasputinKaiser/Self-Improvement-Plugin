# Code of Conduct

## Our standard

This project should be a useful place to discuss local agent harnesses, memory systems, evals, and developer tooling.

Participants are expected to:

- be direct and respectful
- stay on topic
- assume good faith unless there is a clear reason not to
- avoid harassment, threats, insults, or personal attacks
- avoid posting private data, secrets, transcripts, or screenshots with sensitive information

## Unacceptable behavior

Unacceptable behavior includes:

- harassment or abuse
- threats or intimidation
- slurs or personal attacks
- publishing someone else's private information
- spamming issues or pull requests
- intentionally derailing technical discussion

## Enforcement

Maintainers may remove comments, close issues, block users, or limit participation when behavior makes the project harder to maintain.

## Reporting

If you see a problem, contact the maintainer through GitHub. Do not post private information in public issues.
